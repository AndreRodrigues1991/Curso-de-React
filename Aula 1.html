O que é React?

1 - Uma biblioteca JS para criação de interfaces.

Vamos conseguir fazer a criação de HTML, fazer a inserção de CSS e as manipulações da view pra fazer animações, mostra forme, remove formulários

2 - Utilizado para construir SPAs(Single Page Aplication).

É aquela questão de ter 1 arquivo só de index e rodar toda aplicação (é oque está sendo muito utilizado no momento), que gerenciaa um arquivo JavaScript que podem chamar milhares outros arquivos JavaScript pra poder criar uma aplicação tudo baseado no JavaScript.

3 - Baseado em Componentes.

Vamos divir em "quebra cabeça" o nosso site e vamos poder reaproveitar essas peças em diversas partes. um exemplo claro e e nossa barra de componentes

4 - Utiliza o JSX para renderizar o HTML.

5 - E aplica o Virtual Dom para realizar as auterações de DOM.

6 - Podemos adicionar a um projeto ou criar um projeto com ele.
