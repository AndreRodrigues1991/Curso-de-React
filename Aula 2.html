Como instalar o React

1 - Para instalar o React vamos utilizar uma ferramenta chamada Create React APP.

2 - Recebemos todos os arquivos da biblioteca e temos como executá-la

3 - Para utilizar precimos do Node e também npm.

4 - Esta ferramenta também otimiza o app gerado para produção.

5 - É possivel iniciar a aplicação com npm start
